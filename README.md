# ASCEND · Migration Path Planning

Mock interactivo de planificación de migración (conectividad, alcance, plan por IA y autorización de pipeline).

## Archivos

- **`migration-path-planning.html`** — Pantalla principal del flujo; ábrela en el navegador como entrada del proyecto.
- **`Puntos clave migración conclusión (5).html`** — Documento complementario de puntos clave.
- **`Icono bombilla 24dp (2).svg`** — Icono del asistente flotante.
- **`subir-a-github.sh`** — Ayuda para publicar el repo (ajusta la ruta `cd` dentro del script si la usas).

## GitHub Pages

No hay `index.html` en la raíz: la entrada del sitio es **`migration-path-planning.html`**. Tras publicar, abre `https://TU_USUARIO.github.io/NOMBRE_REPO/migration-path-planning.html` (o configura en tu host una redirección raíz → ese archivo).

En el repositorio: **Settings → Pages** → Branch **main**, carpeta **/ (root)**.
